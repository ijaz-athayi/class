how to change the cursor while tap the link
  1. first insert a link in your website using ancor tag
  2. in the css file open ancor atribute
  3. and type "cursor: none;" this will remove the cursor while tapping the link
  4. then write "cursor: url("cursor2.svg"),auto;" to insert a another cursor on it
  5. but it doesnt support png file so use .svg file and that file must place in your folder which this project saved
